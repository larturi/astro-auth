# Astro Auth

## Technologies

- Astro 5.1.5
- Node 20.12.2
- Tailwind CSS
- Login con Google (Firebase)
- Login nativo
- Link de confirmación de registro por email
- Protección de rutas

## Install dependencies

```bash
bun install
```

## Start in Dev Mode

```bash
bun run dev
```

## Build to Production

```bash
bun run build
```
